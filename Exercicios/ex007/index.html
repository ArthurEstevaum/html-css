Texto marcado

Podemos criar também um trecho de texto que será marcado para ter destaque.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno(a tag <big> se tornou obsoleta, mas a tag <small> não.)

Texto excluido

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Usamos o texto inserido para dar uma ênfase e indicar que ele foi adicionado depois.

também existe a tag <u>(não semântica) para sublinhar um texto

Usamos o texto subscrito para algo como H2O com a tag <sub>

Texto sobrescrito
